## for
A Python for loop can traverse any sequence of items, such as a list or a string.

```Python
>>> words = ["are", "you", "good", "now"]  #list
>>> for a in words:
    print(a)
are
you
good
now

>>> for letter in 'python':  #string
    print(" 当前字母: % s" % letter)
p
y
t
h
o
n
```
